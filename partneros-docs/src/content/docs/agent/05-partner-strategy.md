## Agent Overview

**Partner Strategy** defines Ideal Partner Profiles (ICP), evaluates partners, and designs tier structures.

---

## Skills

| Skill | Description |
|-------|-------------|
| **ICP Definition** | Define ideal partner characteristics |
| **Partner Evaluation** | Score and assess potential partners |
| **Tier Design** | Create Bronze/Silver/Gold structure |
| **Market Analysis** | Analyze partner market opportunity |
| **Competitive Positioning** | Strategy vs. competitors |
