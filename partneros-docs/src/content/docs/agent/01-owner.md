## Agent Overview

**The Owner** is your executive decision-maker. This agent handles high-level approvals, strategic decisions, and final sign-offs on partnerships.

---

## Skills

| Skill | Description | Example Prompt |
|-------|-------------|----------------|
| **Budget Approval** | Approve partner program budgets | "Approve $50K MDF for Q2" |
| **Partnership Sign-Off** | Final approval on strategic partners | "Sign off on Acme Corp partnership" |
| **Strategic Direction** | Define partner program goals | "Set 2025 partner revenue target" |
| **Executive Escalation** | Handle C-level issues | "Escalate to CEO for approval" |
| **Board Communications** | Prepare board updates | "Create board deck for Q4" |
| **Risk Decisions** | Approve high-risk partnerships | "Approve partner with 10% default risk" |

---

## When to Use

Use The Owner when you need:
- Final approval on deals > $50K
- Budget allocation decisions
- Strategic partnership sign-off
- Board-level communications

---

## Example Interactions

### Budget Request
> **You:** "Approve $25K for partner marketing funds Q2"
>
> **The Owner:** "Approved. $25K allocated to Q2 MDF. Expected ROI: 3x via co-marketing activities."

### Strategic Partnership
> **You:** "Sign off on enterprise partnership with TechCorp"
>
> **The Owner:** "After reviewing the business case, I approve the TechCorp partnership. Expected revenue: $500K ARR."
