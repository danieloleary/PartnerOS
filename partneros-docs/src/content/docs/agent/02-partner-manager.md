## Agent Overview

**Partner Manager** is your day-to-day relationship manager. This agent handles partner onboarding, Quarterly Business Reviews (QBRs), and ongoing partner success.

---

## Skills

| Skill | Description | Example Prompt |
|-------|-------------|----------------|
| **Partner Onboarding** | Complete onboarding workflow | "Onboard TechCorp as Silver partner" |
| **QBR Management** | Run quarterly reviews | "Run QBR for Acme Corp Q4" |
| **Relationship Management** | Ongoing partner communication | "Schedule monthly check-in with GlobalSys" |
| **Deal Approval** | Approve deal registrations | "Approve TechCorp deal registration" |
| **Escalation Handling** | Resolve partner issues | "Escalate support issue to VP" |
| **Renewal Management** | Handle partner renewals | "Prepare renewal for 2025" |

---

## When to Use

Use Partner Manager when you need:
- New partner onboarding
- QBR scheduling and execution
- Day-to-day partner communication
- Deal registration approval
- Partner issue resolution

---

## Example Interactions

### Onboarding
> **You:** "Onboard Acme Corp as a new Gold partner"
> 
> **Partner Manager:** "Starting onboarding for Acme Corp... ✓ Agreement sent ✓ Portal access created ✓ Kickoff scheduled for next week."

### QBR
> **You:** "Run QBR for TechCorp covering Q4"
> 
> **Partner Manager:** "QBR complete. Summary: 120% to quota, 3 new logos, Q1 pipeline at $250K."
