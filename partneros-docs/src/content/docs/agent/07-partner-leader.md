## Agent Overview

**Partner Leader** handles board presentations, ROI calculations, and executive-level communications.

---

## Skills

| Skill | Description |
|-------|-------------|
| **Board Decks** | Create quarterly board presentations |
| **ROI Analysis** | Calculate partner program returns |
| **Executive Comms** | Prepare CEO/CFO updates |
| **Strategic Planning** | Long-term program strategy |
| **Performance Reports** | Program performance summaries |
