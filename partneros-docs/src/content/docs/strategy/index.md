# Partner Strategy Templates

Define *why* you need partners and *who* you're looking for.

Strategy templates help you build the foundation for a successful partner program before you start recruiting.

| Defining who to recruit | [Ideal Partner Profile](02-ideal-partner-profile/) |
| Evaluating a specific partner | [3C/4C Framework](03-evaluation-framework/) |
| Selling your program to partners | [Competitive Differentiation](04-competitive-differentiation/) |
| Documenting your strategy | [Partner Strategy Plan](05-strategy-plan/) |
