## Overview

Configure systems for a new partner - portal, deals, pricing, and access.

---

## Steps

1. **Portal Setup**
   - Create partner account
   - Assign tier access
   - Configure dashboards

2. **Deal Registration**
   - Enable deal registration
   - Set protection window (90 days)
   - Configure approval workflow

3. **Pricing**
   - Set partner discount tier
   - Configure deal registration pricing
   - Enable quote generation

4. **Access**
   - Grant CRM access
   - Enable reporting
   - Setup notifications
