## How to Use This Template

**Purpose:**
This template structures quarterly business reviews with partners.

**Steps:**

1. **Prepare** — Gather data and metrics (1 hour)
2. **Present** — Review performance (1 hour)
3. **Plan** — Set next quarter goals (1 hour)
